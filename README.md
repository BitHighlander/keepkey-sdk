# KeepKey SDK

Making KeepKey integrations easy

* Update firmware with [updater app](https://beta.shapeshift.com/updater-download)
* Download the latest [KeepKey Client](https://github.com/keepkey/keepkey-client/releases/latest)


### Debugging device

## User onBoarding
